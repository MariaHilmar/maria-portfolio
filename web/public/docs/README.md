# Docs espelhados no site

Cópias de leitura do hub para o site Astro servir em `/docs/`.

Ao atualizar o case study ou competências na raiz `docs/`, copie novamente:

```powershell
Copy-Item ..\..\docs\case-study-juris-sync.md .
Copy-Item ..\..\docs\competencias.md .
```

(Execute a partir de `web/public/docs/`.)
